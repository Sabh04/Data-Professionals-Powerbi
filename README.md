# Data-Professionals-Powerbi
Interactive Power BI dashboard analyzing survey results of data professionals.

💼 Data Professionals Survey Analysis – Power BI Dashboard Project

Overview

This project analyzes a survey of 630+ data professionals to explore job satisfaction, salaries, career switches, and key factors like work-life balance and programming preferences. The interactive Power BI dashboard allows users to explore trends across roles, industries, and demographics, making it ideal for HR analytics, career insights, or data visualization portfolios.
________________________________________

Project Goal

Understand trends in the data profession by analyzing:
•	Job titles, salary ranges, and industry distribution
•	Happiness levels across work-life balance, coworkers, management, mobility, and learning
•	Career switch trends and challenges entering data roles
•	Demographics (age, gender, education, ethnicity, country)
•	Priorities for new jobs (remote work, salary, growth opportunities)

🔍 Key Insights

•	Top Roles: Data Analyst most common, followed by Data Scientist/Engineer
•	Salary Insights: Most respondents earn 0–40k or 41–65k USD; higher salaries correlate with tech/finance industries and advanced degrees
•	Happiness Levels: High for work-life balance (~7/10) and coworkers (~7/10); lower for upward mobility (~5/10) and salary (~5/10)
•	Career Switches: ~70% switched into data roles; many found the transition “Difficult” or “Very Difficult”
•	Job Priorities: “Better Salary” and “Remote Work” are top factors for new opportunities
•	Demographics: Predominantly male (80%), aged 25–35, from US/India/Canada, mostly with Bachelor’s/Master’s degrees
All insights are visualized interactively in Power BI using charts, cards, slicers, and heatmaps.

▶️ How to Use

1.	Download or clone this repository
2.	Open Power BI - Final Project.xlsx in Excel to explore raw survey data
3.	Open Data-Professionals-Dashboard.pbix in Power BI Desktop (free):
o	Refresh data if needed
o	Use slicers to filter by role, country, or salary range
o	Explore pages: Overview, Happiness Metrics, Demographics, Career Insights
4.	Interact with visuals and drill-downs for deeper insights
